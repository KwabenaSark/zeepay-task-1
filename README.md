# Job Application Tracker

A Next.js web application 

Hosted on [https://boisterous-sunshine-346924.netlify.app]

---

## Tech Stack

- **Framework:** Next.js (React)
- **API:** Next.js API Routes (`/api/items`) 
- **HTTP Client:** Axios
- **Data Store:** `data.json`

---

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash

npm install
```

### Running the App

```bash
npm run dev
```

Open [localhost +port](http://localhost:3000)  in your browser.
